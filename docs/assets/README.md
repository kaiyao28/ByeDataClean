# docs/assets/

Visual assets for ByeDataClean — banner image, diagrams, and screenshots.

---

## Banner specification

**File:** `byedataclean_banner.png`  
**Dimensions:** 1280 × 640 px (2:1 ratio, optimised for GitHub README and social cards)  
**Format:** PNG, transparent or light background

### Copy

```
ByeDataClean
From messy data to trustworthy decisions.

Profile  →  Clean  →  Validate  →  Report
```

### Colour palette

| Role | Hex | Use |
|---|---|---|
| Primary blue | `#2b6cb0` | Title text, borders |
| Teal accent | `#319795` | Icons, arrows, highlight boxes |
| Light blue bg | `#e8f1ff` | Left panel (raw data) |
| Light green bg | `#e9ffe8` | Right panel (PASS output) |
| Warning amber | `#d69e2e` | Warning highlights |
| Error red | `#c53030` | BLOCKER highlights |
| Background | `#f7fafc` | Page / card background |

### Layout

Three-panel horizontal layout:

```
┌─────────────────────────────────────────────────┐
│  [left panel]          [centre]    [right panel] │
│                                                  │
│  Raw CSV               →→→         Scorecard     │
│  (messy, red flags)  arrows       (PASS / clean) │
│                                                  │
│  ByeDataClean                                    │
│  From messy data to trustworthy decisions.       │
│                                                  │
│  Profile · Clean · Validate · Report             │
└─────────────────────────────────────────────────┘
```

Icons (Flaticon / Noun Project / inline SVG):
- Left panel: spreadsheet with red ✗ cells
- Arrow: broom sweeping left-to-right
- Right panel: checklist with green ✓ ticks

Font: Inter (or system-ui) — 600 weight for headline, 400 for sub-copy

## Canva AI image prompt

Use this prompt in **Canva → AI Image Generator** (or Figma / Midjourney):

> A clean, modern software product banner. Title reads "ByeDataClean" in bold sans-serif. Subtitle reads "From messy data to trustworthy decisions." Three-panel layout: left panel shows a messy spreadsheet with red error icons; centre shows a stylised broom icon sweeping data clean; right panel shows a tidy checklist report with green checkmarks. Colour palette is soft blue (#2b6cb0), teal (#319795), and light grey (#f7fafc). Rounded corners, flat design, professional and minimalist. 1280x640 pixels.

## Mermaid diagrams

GitHub renders Mermaid diagrams natively inside `.md` files — no image export needed.

The main workflow diagram lives in [README.md](../../README.md#how-it-works).  
Per-step cleaning flowcharts are generated by `--flowchart` at runtime and saved to `reports/cleaning_logs/`.
